# Discord Soundboard MP3 Maker

A lightweight, fast desktop application to create short MP3 clips from YouTube videos, perfect for Discord soundboards.

## Features
- Fast extraction using `yt-dlp` and `ffmpeg`.
- Custom clip duration (up to 15 seconds).
- Modern, dark-themed GUI.
- Automatic folder opening after download.

## Prerequisites
- `ffmpeg` must be installed and in your system PATH.

## How to Use
1. Paste a YouTube URL.
2. Set your start time (in seconds).
3. Set your desired duration (in seconds).
4. Click "Create Clip".

## Installation (for developers)
`pip install -r requirements.txt`
